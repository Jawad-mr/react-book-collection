# Book Library Collection

A React app showcasing a collection of books using reusable components, built with Vite.

## Components
- **BookCard:** Reusable component to display individual book information.

## Links
- **GitHub Repository:** https://github.com/Jawad-mr/react-book-collection.git
- **Live Website:** https://react-book-collection.vercel.app/

## Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Jawad-mr/react-book-collection.git
````

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start development server:

   ```bash
   npm run dev
   ```
